Excel Configuration Template
============================

The Pricing configuration template (Pricing_Config_Template.xlsx) is available at:
    modules/pricing/docs/Pricing_Config_Template.xlsx

This template includes:
- Instructions sheet with methodology comparison
- Settings sheet for global configuration
- VanWestendorp sheet for PSM settings
- GaborGranger sheet for demand curve settings
- Validation sheet for data quality rules

All sheets include:
- Color coding (yellow = required, green = optional, blue = examples)
- In-sheet documentation and field descriptions
- Example values for each setting

For complete template usage documentation, see:
    modules/pricing/docs/USER_MANUAL.md (Section 5: Configuration Reference)

The template was consolidated from the authoritative source:
    docs/Pricing_Config_Template_Manual.md (v11.0)
