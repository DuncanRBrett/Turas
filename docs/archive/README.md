# Archived Documentation

This directory contains historical documentation, specifications, and review materials that have been superseded by current documentation but are kept for reference.

## Subdirectories

### development_specs/
Original development specifications for TURAS modules:
- Conjoint, Key Driver, and Pricing module specs
- Parser v2 enhancement and implementation docs
- Architecture diagrams and checklists

### review_packages/
Code review packages and implementation summaries:
- Key driver code review package
- Pricing module review and implementation summary

### feedback/
Historical AI-assisted code review feedback:
- ChatGPT/Claude reviews for all modules
- Improvements from these reviews have been incorporated into the codebase

### old_manuals/
Older .docx versions of user manuals superseded by comprehensive markdown versions

### tracker_specs/
Historical tracker module development briefs and specifications

## Current Documentation

For current, active documentation, see:
- `/docs/USER_MANUAL.md` - Main TURAS user manual
- `/docs/ARCHITECTURE.md` - Current architecture documentation
- `/docs/TROUBLESHOOTING.md` - Troubleshooting guide
- `/docs/MAINTENANCE.md` - Maintenance guide
- Module-specific docs in `/modules/[module_name]/`

## Note

Files here are archived for historical reference. When in doubt, use the current documentation in `/docs/` and module directories.
